Top 10 music genres in top 5 travel destinations

All the following visualizations were generated using a custom python script that can be found here.

This Python script uses the Spotipy library and Spotify API to analyze the top playlists in the music category "toplists" for the top 5 travel destinations: France, Spain, Italy, United States, and Japan. It extracts the genres of the songs in each playlist and generates a pie chart and a word cloud visualization of the most common genres for each destination. The pie chart shows the percentage of each top genre while the word cloud displays the frequency of each genre as a size of a word in a cloud-like shape. The resulting charts and visualizations are saved as SVG files.

I chose to create my own visualizations using the Spotify Python library because I am passionate about music and enjoy traveling to different countries. The library allowed me to generate pie charts and word clouds that showcase the top genres in the music playlists of five popular travel destinations. I particularly like the word clouds because they present the genre information in a visually appealing way.
